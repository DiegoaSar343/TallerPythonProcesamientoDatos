# TallerPythonProcesamientoDatos
En este repositorio se encuentra un taller asignado en la clase de Procesamiento de datos, en el encontrarán diferentes notebooks con explicaciones y ejercicios de Python básico
Cada archivo esta titulado con Python seguido del tema que trata el notebook, en el se halla la explicacion de como usar los diferentes temas, con operaciones y ejemplos, para al final tener un breve examen para el alumno.
Aparte de los ejercicios, también se encuentra un archivo titulado con bonus, el cual es un notebook que esta diseñado para que el estudiante pueda practicar lo visto en los otros notebooks, haciendo uso de los diferentes conocimientos aprendidos.
